# 市占率堆积柱形图 — 可视化项目

## 项目说明

基于各省份/区域各车型市占率数据，生成堆积柱形图，用于本品与竞品的市场份额对比分析。

## 文件清单

| 文件 | 说明 |
|------|------|
| `chart_stacked_bar.py` | 可视化主脚本（配置驱动，支持多本品） |
| `数据.xlsx` | 数据源（省份 + 本品 + 竞品市占率） |
| `市占率堆积柱形图_高山.png` | 本品=高山的输出图表 |

## 图表设计逻辑

1. **堆积柱形图**：每个柱子代表一个区域（省份/全国合计），柱内按市占率从大到小从上往下排列
2. **本品对比突出**：本品以鲜明红色标记，竞品以低饱和度冷灰色系弱化
3. **一致性**：同一车型在所有省份中使用相同颜色
4. **排名标注**：X轴下方标注本品在各区域的市占率排名
5. **全国合计**：以斜线填充柱区分，作为参考基准
6. **全标签显示**：所有车型的数据标签均会显示，小份额自适应字号

## 使用方法

### 1. 安装依赖

```bash
pip install pandas matplotlib openpyxl numpy
```

### 2. 准备数据

Excel 格式要求：
- 第一列：`省份`（区域名称，包含"全国合计"行作为汇总基准）
- 其余列：各车型市占率（小数形式，如 0.09 = 9%），本品列名与配置一致即可

### 3. 运行脚本

```bash
python chart_stacked_bar.py
```

### 4. 切换本品

编辑脚本顶部 `ANALYSES` 配置：

```python
ANALYSES = [
    dict(
        product='高山',               # ← 改这里
        data_file='数据.xlsx',         # ← 改这里
        sheet_name=0,
        output_file='市占率堆积柱形图_高山.png',
        title='各省份车型市占率分布 — 本品：高山',
    ),
    # 多本品示例：取消注释即可
    # dict(
    #     product='蓝山',
    #     data_file='数据_蓝山.xlsx',
    #     sheet_name=0,
    #     output_file='市占率堆积柱形图_蓝山.png',
    #     title='各省份车型市占率分布 — 本品：蓝山',
    # ),
]
```

### 5. 配置项说明

| 参数 | 说明 |
|------|------|
| `product` | 本品在 Excel 中的列名 |
| `data_file` | 数据文件路径 |
| `sheet_name` | Sheet 索引（0=第一个）或名称 |
| `output_file` | 输出 PNG 路径 |
| `title` | 图表标题 |

### 6. 颜色自定义

```python
PRODUCT_COLOR = '#E63946'       # 本品颜色（鲜明暖色）
COMPETITOR_PALETTE = [...]       # 竞品调色板（冷灰色系）
```

## 兼容性说明

- Python >= 3.8
- Windows / macOS / Linux 均支持
- 自动检测系统中文字体（Microsoft YaHei / SimHei / Noto Sans CJK SC）
- 竞品数量不限，颜色从调色板自动轮替分配
